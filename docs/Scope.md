# Project Scope Statement

## Project Name

Sentinel Spectrum Technologies Website

---

## Project Purpose

The purpose of this project is to design and develop a professional two-page website for Sentinel Spectrum Technologies, a fictional defense contracting company specializing in Software-Defined Radio (SDR) solutions. This project demonstrates my ability to combine project planning, documentation, version control, and web development into a complete digital project.

The finished website will also serve as a portfolio piece that reflects my interests in electronic warfare, secure communications, and RF technology while applying the project management principles learned throughout this course.

---

## Project Description

This project consists of a two-page website developed using HTML and CSS and published through GitHub Pages.

The homepage (`index.html`) will introduce Sentinel Spectrum Technologies, its mission, and the SDR services it provides. Visitors will be able to learn about the company's focus on secure communications, SDR integration, technical consulting, and defense technology solutions.

The second page (`about.html`) will provide background information about the company, its values, industries served, and the importance of Software-Defined Radio technology.

A single external stylesheet (`style.css`) will provide consistent formatting across both webpages.

The GitHub repository will also contain the required project documentation:

- README.md
- scope.md
- plan.md
- retrospective.md

---

## Desired Results

The project will be considered successful when:

- The website is successfully published through GitHub Pages.
- Both webpages display correctly on desktop and mobile devices.
- Navigation between pages functions correctly.
- The website presents a professional appearance.
- All required documentation is completed.
- The GitHub repository is organized according to the assignment requirements.
- The project demonstrates proper use of HTML, CSS, Git, and GitHub.

---

## Exclusions

The following items are outside the scope of this project:

- User accounts or login systems
- Databases
- Online shopping functionality
- Contact forms with backend processing
- JavaScript functionality
- Server-side programming
- Real defense contractor information
- Classified or sensitive military information
- Additional webpages beyond those required by the assignment
